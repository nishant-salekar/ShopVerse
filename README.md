🛍️ ShopVerse – Shopping Platform

ShopVerse is a full-stack e-commerce / shopping platform built with MERN STACK.  
It provides a seamless user experience for browsing products, managing carts, checkout/payment,  
and admin/product-management tools.

---

🚀 Features (Core + Key Functionalities)

🛒 Product Browsing & Search – View product listings, filter/sort by category, search products.  
🧑‍💻 User Authentication & Profile – Sign up / log in / secure authentication.  
🛍️ Shopping Cart & Checkout – Add to cart, review cart, and checkout.  
💳 Payments Integration – Secure payments for orders.  
📦 Order Management / Purchase History – View past orders, track status.  
🔧 Admin Tools – Add / edit / delete products, manage inventory, view orders.  
📱 Responsive UI – Clean layout, mobile-friendly, great UX.

---

🛠️ Tech Stack

**Frontend**
- React  
- Tailwind CSS / Styled-Components  
- State management (Redux / Context API)  
- Axios / Fetch API  

**Backend**
- Node.js + Express  
- MongoDB + Mongoose  
- Authentication  
- Payment Gateway Integration  

**Deployment**
- Vercel

⚙️ Setup Instructions

1️⃣ Clone the repository  

2️⃣ Install dependencies  
**Frontend**  
cd client  
npm install  

**Backend**  
cd server  
npm install  

3️⃣ Configure environment variables  
Paste your variables in `.env` files in server and client folders.

4️⃣ Run the project  
**Backend**  
cd server  
npm run dev  

**Frontend**  
cd client  
npm start
